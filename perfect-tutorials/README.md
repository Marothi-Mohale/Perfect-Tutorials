# Perfect Tutorials

Monorepo for the Perfect Tutorials marketing site and API.

## Apps

- `apps/web`: Next.js marketing site
- `apps/api`: NestJS API

## Scripts

```bash
pnpm dev
pnpm build
pnpm lint
pnpm typecheck
pnpm test
pnpm test:e2e
```

## Environment

Copy values from `.env.example` into your deployment platform or local shell.
